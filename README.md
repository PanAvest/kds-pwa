# KDS PWA (Next.js 15 + TS + Tailwind + Supabase + Paystack + FCM)

Production-ready scaffold with PWA (next-pwa), push notifications (FCM), Supabase auth/DB, and Paystack stubs.
See the chat message for the full quickstart and deployment steps.

Note: On iOS, the Capacitor shell is an access-only companion; purchases happen on web/Android or other platforms outside the iOS app.

KDS Learning – Privacy Policy (English – U.K.)
Effective Date: 27 November 2025
KDS Learning (“we”, “our”, “us”) is operated by PanAvest International & Partners.
We are committed to protecting your privacy and ensuring that your personal information is handled responsibly and transparently.
This Privacy Policy explains how our PWA/mobile app collects, uses, stores, and safeguards your data.
1. Information We Collect
We collect only the data necessary to operate the KDS Learning platform and deliver certified learning services.
1.1 Account Information
Full name
Email address
Password (securely encrypted via Supabase Auth)
Profile information you choose to update
1.2 Course & Learning Data
Courses you enrol in
Exam attempts, scores, and progress
E-book access history
Certificates generated and downloads
1.3 Payment Information
Payments are handled securely by Paystack.
We do not store card numbers or sensitive financial data.
We may receive:
Payment reference
Transaction status
Date/time of purchase
1.4 Device & Technical Data
Automatically collected for app functionality:
Device type and operating system
App version
IP address (for security)
Error logs and crash data
1.5 Optional Data
If enabled by the user:
Push notification token (for course updates and exam alerts)
2. How We Use Your Data
We use your data strictly for:
Creating and managing your learning account
Delivering purchased courses, exams, and e-books
Generating CPPD-recognised certificates
Processing payments securely
Improving the app’s performance and content
Providing support and responding to enquiries
Ensuring security, fraud prevention, and compliance
We never sell your data.
3. Data Sharing
We only share data with trusted service providers required to operate the platform:
3.1 Supabase
Authentication
Database storage
Secure file (PDF/e-book) delivery
3.2 Paystack
Payment processing
3.3 Apple, Google, and Capacitor Plugins
Used only for:
Native browser views
In-app navigation
Security and crash analytics
These providers have their own privacy policies and comply with international security standards.
4. Data Protection & Security
We implement a multi-layered security approach:
End-to-end encrypted authentication (Supabase Auth)
Row Level Security (RLS) restrictions on user data

Secure payments through PCI-DSS-compliant Paystack
Access limits to authorised systems only
We take reasonable steps to keep your information secure and confidential.
